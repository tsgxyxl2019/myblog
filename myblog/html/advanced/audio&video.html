需求:

引入视频或音频

Test_area：

1.音频播放

2.视频播放

3.多选格式视频播放

4.显示部分视频内容，多余部分溢出隐藏

知识点：audio&video

引入音频与视频的标签。属于h5新功能

audio标签表示嵌入音频文件，video标签表示嵌入视频文件。默认控件是不显示的，可通过controls属性来显示控件。

为了能够支持多个备选文件的兼容支持，可以配合source标签。

loop重复播放 autoplay自动播放

source作用：当浏览器不支持文件格式的时候，source可添加备选播放源，如有一合适，停止向下选择过程

外部添加div 溢出隐藏 高度为部分视频高度, 视频样式为最小宽高100%，确保视频不会被等比缩小

html源代码：


<p>1.音频播放</p>
<audio style="margin: 0 50px;" src="./html_audio_video/johann_sebastian_bach_air.mp3" controls loop autoplay></audio>
<p>2.视频播放</p>
<video style="margin: 0 50px;" src="./html_audio_video/Intermission-Walk-in_512kb.mp4" controls></video>
<p>3.多选格式视频播放</p>
<video style="margin: 0 50px;" controls>
    <source src="./html_audio_video/Intermission-Walk-in.ogv"></source>
    <source src="./html_audio_video/Intermission-Walk-in_512kb.mp4"></source>
</video>
<p>4.显示部分视频内容，多余部分溢出隐藏</p>
<div id="videobox" style="margin: 0 50px; width: 100%; height: 300px; overflow: hidden;">
    <video style="min-width: 100%; min-height: 100%;" src="https://shimmer.neusoft.edu.cn/h/www/media/intro2016.mp4" autoplay="" muted ></video>
</div>
