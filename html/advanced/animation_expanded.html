需求:

1.animation-fill-mode 实现不同的动画

2.animation-direction 实现不同的动画方向

3.利用animate动画库 实现不同的动画

Test_area：

1.animation-fill-mode 实现不同的动画

1.animation-fill-mode: none
2.animation-fill-mode: backwards
3.animation-fill-mode: forwards
4.animation-fill-mode: both

2.animation-direction 实现不同的动画方向

1.animation-direction: normal
2.animation-direction: reverse
3.animation-direction: alternate

3.利用animate动画库 实现不同的动画

知识点：

animation-fill-mode : 规定动画播放之前或之后，其动画效果是否可见。
none (默认值) : 在运动结束之后回到初始位置，在延迟的情况下，让0%在延迟后生效
backwards : 在延迟的情况下，让0%在延迟前生效
forwards : 在运动结束的之后，停到结束位置
both : backwards和forwards同时生效

animation-direction : 属性定义是否应该轮流反向播放动画。
normal (默认值) : 永远都是正向 , 从0%~100%
reverse : 永远都是反向 , 从100%~0%
alternate : 一次正向(0%~100%)，一次反向(100%~0%)

一款强大的预设css3动画库。
官网地址：https://daneden.github.io/animate.css/
基本使用：
animated : 基类(基础的样式，每个动画效果都需要加)
infinite : 动画的无限次数

css源代码：


<style>
    .part_1 div, .part_2 div{width: 100px;height: 100px;background: red; animation: part_1_move 3s linear;margin-bottom: 20px;color: white;}
    .part_2 div{animation-iteration-count: infinite;}
    .part_1 .box1{animation-delay: 2s;animation-fill-mode: none;}
    .part_1 .box2{animation-delay: 2s;animation-fill-mode: backwards;}
    .part_1 .box3{animation-delay: 2s;animation-fill-mode: forwards;}
    .part_1 .box4{animation-delay: 2s;animation-fill-mode: both;}
    @keyframes part_1_move {
        0%{transform: translate(0px,0px);background: blue;}
        100%{transform: translate(300px,0px);}
    }
        
    .part_2 .box1{animation-direction: normal}
    .part_2 .box2{animation-direction: reverse}
    .part_2 .box3{animation-direction: alternate}
</style>
