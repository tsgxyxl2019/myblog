知识点：

添加单线 : border-collapse : collapse
隐藏空单元 : empty-cells : hide
斜线分类 : border / rotate
列分组 : colgroup / col

border 利用加粗上边框和左边框添加斜线，再绝对定位文字至单元格中

colgroup 可直接针对整列进行操作
